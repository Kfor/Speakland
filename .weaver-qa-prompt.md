3. **验证项目可构建** — 执行 `npx expo export --platform web` 确认 web 端可成功构建
4. **端口冲突检测** — 启动任何 server 前先 `lsof -i :<port>` 检查。**不要 kill 其他进程**。
5. **检查 TypeScript 编译** — 执行 `npx tsc --noEmit` 确认无类型错误

9. **检查 Edge Function** — 确认 llm-proxy 实现了 JWT auth、streaming、non-streaming
