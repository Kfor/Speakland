# Story References - GenZ 热门 IP 剧情梗概

本目录包含从 GenZ 热门影视/动漫中提取的剧情梗概，用于 Speakland RPG 剧情树的内容创作参考。

每个故事已按 Speakland 的 Story → Segment → Branch 结构做了初步拆解，
方便后续直接转化为 seed data 或通过 CMS 录入。

## 选题标准

1. **GenZ 热度验证**：TikTok/社交媒体高讨论度，流媒体平台高播放量
2. **对话密度高**：剧情推进依赖角色对话而非纯动作，适合语言学习场景
3. **世界观清晰**：场景和角色可以直接提取，无需复杂背景知识
4. **语言难度适配**：覆盖初级到中级，不同 IP 对应不同学习阶段
5. **文化吸引力**：年轻用户一看就有代入感和兴趣
